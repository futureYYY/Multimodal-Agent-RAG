1.进入docker官网：https://docs.docker.com/    下载Docker Desktop Installer.exe
2.在想安装的盘符下创建文件夹 例如:F: 创建了installers 文件夹
3.把下载Docker Desktop Installer.exe 文件复制到F:盘符 创建的installers 文件夹下
4.执行命令： .\"Docker Desktop Installer.exe" install --installation-dir="F:\Program Files\Docker"  实现自动安装，如有弹窗点确定就行
5.wsl -l -v 查看docker里面的wsl或者自己单独安装的wsl版本
6.现在，请继续执行我们“阶段四”的计划。

启动 Docker Desktop:
如果它还没运行，请从开始菜单启动它。
进入设置:
点击 Docker Desktop 窗口右上角的 齿轮图标 (⚙️)。
定位到数据配置:
在左侧菜单中，选择 Resources -> Advanced。
修改磁盘镜像位置 (Disk image location):
您会看到 Disk image location 这个选项，它当前可能指向一个 C: 盘的路径。
点击 Browse 按钮。
导航到您在 F: 盘上为 Docker 数据准备的文件夹，例如 F:\installers\DockerData (如果不存在，可以现在创建它)。
选择该文件夹。
应用并重启:
点击右下角的 Apply & restart 按钮。

7.打开命令提示符以管理员方式运行 再安装 wsl： wsl --install
8.安装完启动命令：wsl.exe -d Ubuntu
